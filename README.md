💰 Automated Expense Management

A Python-based command-line application that helps users track and manage their daily expenses efficiently. The project allows users to add expenses, view expense records, and calculate total spending through a simple menu-driven interface.

📖 Project Overview

Managing personal expenses is an important part of financial planning. This application provides a simple way to:

Record expenses
Categorize spending
Track daily transactions
View spending history
Calculate total expenses

The project is designed for beginners learning Python fundamentals and demonstrates practical use of data structures and control flow.

✨ Features

✅ Add new expenses

✅ Store expense details:

Date
Category
Description
Amount

✅ View all recorded expenses

✅ Calculate total expenses

✅ User-friendly menu-driven interface

🚀 Getting Started
Prerequisites

Make sure Python 3 is installed on your system.

Check installation:

python --version
Clone the Repository
git clone https://github.com/AnkitRai991/Automated-Expense-Management.git
Navigate to Project Directory
cd Automated-Expense-Management
Run the Application
python expense_tracker.py
📋 Application Menu

1. Add Expense
2. View All Expenses
3. View Total Expenses
4. Exit
